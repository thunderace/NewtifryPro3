Ce qu'il faut

  • Un SenderId pour s'enregistrer auprès du service FCM de Google.
  • Une APIKey pour envoyer des messages à votre périphérique
  • L'ID FCM(DeviceID) de votre périphérique

Obtenir le SenderId et l'APIKey

Une partie de ce guide est issue de la documentation Google présente ici

Création d''un projet Google API

Pour créer un projet Google API:

  • Ouvrez la console Google APIs.
  • Si vous n'avez pas encore créé de projet, cette page va vous demander de le faire:

Note: Si vous avez déjà un projet existant, la première page que vous verrez sera le Dashboard. De cette page vous pouvez créer un nouveau projet en ouvrant le menu projet (en haut à gauche) et en choisissant Other projects > Create.

  • Cliquez sur Create project. L'URL de votre browser va prendre la forme suivante:

https://code.google.com/apis/console/#project:4816662342

  • Notez la valeur située après #project: (***4816662342*** dans cet exemple). Cette valeur est votre ID de projet et sera utilisée en tant que SenderId.

Activer le service FCM

Pour activer le service FCM:

  • Dans le menu de gauche, cliquez sur Services.
  • Activez le Google Cloud Messaging for Android.
  • Acceptez les conditions d'utilisation

Obtenir l 'APIKey

Pour obtenir l'APIKey:

  • Dans le menu de gauche, cliquez sur API Access. Vous verrez apparaitre un écran semblable à celui-ci:

  • Cliquez sur Create new Server key. L'écran suivant apparait:

  • Cliquez sur Create: Notez la valeur de API key (***YourKeyWillBeShownHere*** dans cet exemple).

Enregistrer votre périphérique

Pour enregistrer votre périphérique, dans NewtifryPro:

  • Ouvrez le menu
  • Selectionnez 'Enregistrement'
  • Entrez le SenderID obtenu précédemment
  • Attendez la fin de l'enregistrement
  • Ouvrez le menu
  • Selectionnez 'Paramètres'
  • Selectionnez 'Test'
  • Après une durée variable, vous recevrez un message de test

Vous êtes maintenant prêt à recevoir des messages!

Envoyer des messages à un périphérique

Vous avez besoin de votre APIKey et du DeviceID de votre périphérique

Obtenir le DeviceID du périphérique

Dans NewtifryPro:

  • Ouvrez le menu
  • Selectionnez 'Paramètres'
  • Selectionnez 'Identifiant de l'appareil'
  • Entrez votre adresse mail et effectuez l'envoi

Une fois l'ID de l'appareil obtenu, vous pouvez utiliser le script php disponible ici : NewtifryPro script

Vous pouvez envoyer des messages à votre périphérique!
